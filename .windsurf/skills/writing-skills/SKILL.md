# How to
- Create .windsurf/skills/<skill-name>/SKILL.md
- Frontmatter:
  - name: lowercase with hyphens
  - description: when to invoke and purpose
- Body: succinct checklist tailored to TAC (types, query keys, errors, UI tokens, tests)

# Best practices
- Keep it short and action-oriented
- Reference repo conventions in AGENTS.md
- Add project-specific commands (typecheck/lint/tests)

# Exit criteria
- Skill is discoverable and useful to invoke automatically or via @mention.
